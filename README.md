
# Название проекта

Wind Dataset ML model 

## Описание 

Данный проект был сделан для предсказания скорости ветра с помощью модели машинного обучения с применение линейной регрессии. 

## Дата сет

Данный дата сет был взят с сайта Kaggle.

* Количество столбцов – **9** 
* Количество строк – **6574** 

Дата сет содержит такие столбцы как: 

1. DATE
2. WIND
3. IND 
4. RAIN
5. IND.1
6. T.MAX
7. IND.2
8. T.MIN
9. T.MIN.G

## Методы, подход

Была произведена:
- Заполнены пропуски средними значениями
- Колонка DATE разбит на колонки day, month, year
- Разделение данных на тренировочные и тестовые 
- Нормализация значений 
- Создание функций predict и gradient
- Получение предсказания

## Полученные данные

На датасете предсказания скорости ветра MSE ≈ 21.8

## Основные выводы

* Линейная модель плохо справляется из-за нелинейности данных, шума, сложных зависимостей. Для улучшения нужны более сложные модели
